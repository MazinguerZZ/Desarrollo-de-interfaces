**1.** Investiga qué es `prototype` y explica con tus palabras qué es y para qué sirve.

**2.** Explica qué significa este tipo de líneas en JavaScript:

- Un constructor con propiedades.
- Al menos un método.
- Dos objetos creados con `new` que usen el mismo método.
