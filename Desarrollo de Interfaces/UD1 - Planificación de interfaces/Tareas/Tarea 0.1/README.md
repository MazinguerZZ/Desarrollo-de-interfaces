## Punto de partida

A partir del archivo `html` adjunto, genera un fichero CSS cuyo código comience exactamente por:
```css
body {
  font-family: Arial, sans-serif;
}

.card {
  /* aquí trabajaremos */
}
```

---

## Objetivo

Consigue que la tarjeta cumpla todos estos requisitos:

- Tenga un **borde y una sombra suave**.
- Esté **centrada en la pantalla**.
- La **imagen adaptada** al ancho de la tarjeta.
- Un **botón con color de fondo** y efecto `hover`.
- **Texto jerarquizado:** nombre en grande, precio destacado.

---

## Valorable ⭐

Se valorará positivamente:

- **Redondear esquinas** o darle forma de **etiqueta de venta** *(ver sección How To de W3Schools)*.
- Usar **degradado** (`gradient`) en algún elemento.
- **Animar el efecto `hover`** con transiciones CSS.
