# Coffee Express – Proyecto de Interfaz Web

## Descripción del proyecto

Este proyecto consiste en el diseño de una interfaz web para una tienda de café llamada **Coffee Express**. La web está dividida en varias secciones: Inicio, About Us, Diseños y Contactenos. El objetivo es ofrecer una experiencia visual atractiva, moderna y fácil de usar para los clientes. Se incluyen elementos como reseñas, productos (cafés calientes y fríos), galería de imágenes y un formulario de contacto.

La interfaz está pensada para adaptarse a distintos dispositivos, incluyendo móviles, y puede integrarse con controles por voz o gestos para mejorar la accesibilidad.

## Pasos de ejecución

1. Clonar o descargar el proyecto en tu equipo.
2. Abrir el archivo `index.html` en tu navegador para visualizar la interfaz.
3. Asegurarse de que las imágenes estén en la carpeta `/img` y los estilos en `/css`.
4. Para editar el diseño, modificar los archivos HTML y CSS según sea necesario.
5. (Opcional) Probar en dispositivos móviles para verificar el diseño responsive.
