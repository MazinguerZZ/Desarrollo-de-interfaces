## Conceptos a desarrollar

**1. API** — Consumir datos de un servidor para nuestra app.

**2. Fetch** — Suele estar dentro de un `useEffect` para ejecutar una llamada a una API.

**3. JSON** — Guarda datos de tipo clave-valor.

**4. Asincronía** — Permite usar APIs de forma que, mientras llegan los datos, la app sigue funcionando con normalidad.

**5. Testing**
- *Unitario:* comprueba la validez/funcionamiento de un elemento (nombre, email…).
- *Integración:* comprueba el funcionamiento global.
- Herramientas: **Jest**, **Cypress**, **Playwright**.
